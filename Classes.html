Classes in gaming

Classes are special character archetypes, used in most games.
The purpose of seperating characters by class, is to differentiate the characters combat styles.
Dungeons and Dragons created this concept, which has since been adopted into many different types of gaming, but consistently in RPG games.

A so called 'Holy Trinity' exists in selecting classes for the purpose of creating a party. Usually this consists of:

  1. A Tank: any class that can withstand substantial damage from enemies without being killed. In this case a Paladin.
  2. A DPS (damage per second): any class that can generate substantial damage per second on enemies, in this case a Mage.
  3. A healer: any class that can heal the other members of the team, in this case a Bard.

It is common for most players in any RPG type game to have personal favorite class and to usually play the same class in different games.
